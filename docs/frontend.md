# 前端开发说明

## 技术栈
- Vue.js
- Element UI
- Axios
- Vue Router
- Vuex

## 目录结构
```
vue/
  ├── src/
  │   ├── api/         # 接口请求
  │   ├── components/  # 公共组件
  │   ├── views/       # 页面视图
  │   ├── router/      # 路由配置
  │   ├── store/       # 状态管理
  │   └── assets/      # 静态资源
  └── ...
```

## 开发规范
- 组件化开发，单一职责
- 代码注释清晰，命名规范
- 接口统一管理，错误处理完善
- 响应式设计，兼容主流浏览器

## 启动方式
```bash
cd vue
npm install
npm run dev
```
