# 博客系统设计规范

## 概述

本文档定义了 newblog 博客系统的统一设计规范，包括圆角、阴影、间距、颜色等视觉元素，确保整个博客系统具有一致的视觉体验。

## 设计原则

1. **一致性**: 所有页面使用相同的设计规范
2. **可读性**: 确保内容清晰易读
3. **响应式**: 适配不同设备尺寸
4. **可维护性**: 使用统一的变量和样式

## 圆角规范

### 统一圆角标准

| 元素类型 | 圆角值 | 使用场景 |
|---------|--------|---------|
| **卡片容器** | 16px | 文章卡片、侧边栏组件、统计卡片 |
| **大卡片** | 20px | 主要内容容器、详情页主卡片 |
| **小元素** | 8px | 按钮、标签、徽章 |
| **圆形元素** | 50% | 头像、图标按钮 |
| **图片** | 12px | 文章封面、内容图片 |
| **代码块** | 12px | 代码展示区域 |
| **输入框** | 8px | 表单输入框 |

- 标签/徽章: `20px`

#### 博客首页

- 文章卡片: `12px`

#### 关于页面

## 过渡动画

### 统一过渡标准

| 属性 | 值 | 使用场景 |
|-----|---|---------|
| **快速过渡** | `0.2s ease` | 小元素 |
| **标准过渡** | `0.3s ease` | 卡片、按钮 |
| **慢速过渡** | `0.4s ease` | 大容器 |
| **复杂过渡** | `0.4s cubic-bezier(0.4, 0, 0.2, 1)` | 复杂动画 |

### 常用动画效果

```css
/* 悬停上浮 */
transform: translateY(-4px);

/* 悬停放大 */
transform: scale(1.05);

/* 悬停阴影增强 */
box-shadow: 0 15px 50px rgba(0, 0, 0, 0.15);

/* 淡入 */
opacity: 0 → 1;

/* 上浮淡入 */
opacity: 0;
transform: translateY(30px);
→
opacity: 1;
transform: translateY(0);
```

## 实施指南

### 1. 使用 CSS 变量

在项目中定义统一的 CSS 变量：

```css
:root {
  /* 圆角 */
  --radius-sm: 8px;
  --radius-md: 12px;
  --radius-lg: 16px;
  --radius-xl: 20px;
  --radius-full: 50%;

  /* 阴影 */
  --shadow-xs: 0 2px 8px rgba(0, 0, 0, 0.06);
  --shadow-sm: 0 4px 20px rgba(0, 0, 0, 0.08);
  --shadow-md: 0 6px 25px rgba(0, 0, 0, 0.1);
  --shadow-lg: 0 10px 40px rgba(0, 0, 0, 0.12);
  --shadow-xl: 0 15px 50px rgba(0, 0, 0, 0.15);

  /* 间距 */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 12px;
  --spacing-lg: 16px;
  --spacing-xl: 20px;
  --spacing-2xl: 24px;
  --spacing-3xl: 30px;
  --spacing-4xl: 40px;
  --spacing-5xl: 50px;
  --spacing-6xl: 60px;
  --spacing-7xl: 80px;
  --spacing-8xl: 100px;

  /* 颜色 */
  --color-primary: #667eea;
  --color-secondary: #764ba2;
  --color-success: #67c23a;
  --color-warning: #e6a23c;
  --color-danger: #f56c6c;
  --color-info: #909399;

  --color-text-primary: #333333;
  --color-text-secondary: #666666;
  --color-text-tertiary: #999999;
  --color-border: #e8e8e8;
  --color-divider: #f0f0f0;
  --color-bg: #f5f7fa;
  --color-card: #ffffff;

  /* 过渡 */
  --transition-fast: 0.2s ease;
  --transition-base: 0.3s ease;
  --transition-slow: 0.4s ease;
  --transition-complex: 0.4s cubic-bezier(0.4, 0, 0.2, 1);
}
```
