### 4.2 建议启动方式

#### 后端启动
```bash
# 编译项目
mvn clean install -DskipTests

# 启动后端服务
cd ruoyi-admin
mvn spring-boot:run
```

#### 前端启动
```bash
# 安装依赖
cd ruoyi-ui
npm install

# 启动开发服务器
npm run dev
```

#### Docker 一键部署
```bash
# 开发环境
docker compose -f docker-compose.dev.yml up -d

# 生产环境
docker compose -f docker-compose.prod.yml up -d
```

## 5. 配置文件检查

### 5.1 核心配置文件

#### 后端配置
- **配置文件**: ruoyi-admin/src/main/resources/application.yml
- **数据库配置**: MySQL 8.4，默认端口3306，数据库名newblog
- **Redis配置**: 默认端口6379，无密码（生产环境建议设置密码）
- **服务端口**: 8080
- **Token配置**: 默认密钥已配置，生产环境建议更换为复杂密钥

#### 前端配置
- **配置文件**: ruoyi-ui/.env.development, .env.production
- **API基础路径**: http://localhost:8080
- **构建命令**: npm run build:prod（生产环境）
