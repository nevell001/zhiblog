# 变更日志（Changelog）

## v0.1.0  初始版本
- 完成项目环境搭建，初始化 RuoYi-Vue 框架
- 实现用户注册、登录、权限管理基础功能
- 实现文章的增删改查、分类、标签管理

## v0.2.0  评论与互动
- 增加评论系统，支持多级评论、点赞、审核
- 敏感词过滤功能上线

## v0.3.0  站点与资源管理
- 站点信息配置、友情链接、公告、广告位管理
- 媒体资源上传与管理，支持本地/OSS

## v0.4.0  日志与监控
- 操作日志、登录日志、系统监控功能上线

## v0.5.0  前台展示与优化
- 首页、文章页、分类页、标签页、归档页、个人中心、搜索等页面上线
- 响应式设计，适配PC和移动端

## v1.0.0  正式版发布
- 完善文档、测试、部署脚本
- 性能优化与安全加固
- 支持RSS、API、多语言、夜间模式等扩展功能
