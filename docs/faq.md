# 常见问题（FAQ）

## 1. 依赖下载慢或失败怎么办？
- 建议配置国内 Maven 源（如阿里云）。
- 检查网络连接，必要时多尝试几次。

## 2. 数据库初始化失败？
- 请确认 MySQL 版本与 `sql/` 目录下脚本兼容。
- 检查 `application.yml` 数据库配置是否正确。
- 数据库需提前创建，字符集推荐 utf8mb4。

## 3. 前端/后端端口冲突？
- 默认后端 8080，前端 8000。可在配置文件中修改。
- 检查本机是否有其他服务占用端口。

## 4. 登录后接口报 401/403？
- 检查 Token 是否正确携带在请求头（Authorization: Bearer xxx）。
- 检查用户权限、角色配置。

## 5. 文件上传失败？
- 检查本地存储路径或 OSS 配置。
- 检查文件大小、格式限制。

## 6. 如何部署到生产环境？
- 推荐使用 Docker 部署，详见 `docs/deploy.md`。
- 配置环境变量，关闭调试模式，开启 HTTPS。

## 7. 其他问题如何反馈？
- 请在 Issues 区提交详细描述，附带日志和截图。
