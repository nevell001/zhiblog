## 常见问题

### 1. 菜单不显示

- 检查数据库菜单表是否正确更新
- 检查用户角色是否拥有相应菜单权限
- 清除浏览器缓存

### 2. 页面加载失败

- 检查组件路径是否正确
- 检查API接口是否正常响应
- 查看浏览器控制台错误信息

### 3. 权限验证失败

- 检查权限数据格式是否正确
- 检查权限验证逻辑是否正确
- 重新登录获取最新权限数据

## 技术细节

### 数据库修改

主要修改了`sys_menu`表中的以下字段：
- `path`: 菜单路径
- `component`: 组件路径

确保与前端路由配置一致。

### 前端修改

主要修改了以下文件：
- `/src/components/iFrame/index.vue`: 修复组件导入问题
- `/src/store/modules/permission.js`: 优化权限控制和菜单加载逻辑
- `/src/views/admin/monitor/druid/index.vue`: 修复iFrame组件导入
- `/src/views/admin/tool/swagger/index.vue`: 修复iFrame组件导入
