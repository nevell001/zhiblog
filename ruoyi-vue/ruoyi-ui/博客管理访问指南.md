# 博客管理功能访问指南

## 🔑 登录步骤

### 1. 访问登录页面
```
http://localhost:3000/login
```

### 2. 使用管理员账号登录
- 用户名：admin
- 密码：admin123
- 验证码：需要刷新页面获取

### 3. 登录后访问博客管理
```
http://localhost:3000/admin/blog
```

## 📋 功能菜单

博客管理包含以下功能模块：

### 📝 文章管理
- 路径：`/admin/blog/article`
- 权限：admin, editor
- 功能：文章的增删改查、发布、草稿管理

### 📁 分类管理  
- 路径：`/admin/blog/category`
- 权限：admin, editor
- 功能：文章分类的创建和管理

### 🏷️ 标签管理
- 路径：`/admin/blog/tag`
- 权限：admin, editor
- 功能：文章标签的创建和管理

### 💬 评论管理
- 路径：`/admin/blog/comment`
- 权限：admin, editor
- 功能：评论审核和管理

### 🔗 友链管理
- 路径：`/admin/blog/friendLink`
- 权限：admin
- 功能：友情链接管理

### ⚙️ 博客设置
- 路径：`/admin/blog/setting`
- 权限：admin
- 功能：博客基本设置

## 🔧 API接口列表

所有博客管理API都使用 `/system/*` 路径：

- 文章列表：`GET /system/article/list`
- 分类列表：`GET /system/category/list`
- 标签列表：`GET /system/tag/list`
- 评论列表：`GET /system/comment/list`

## 🛡️ 权限说明

- **admin**：拥有所有权限
- **editor**：拥有文章、分类、标签、评论管理权限
- 未登录用户：自动重定向到登录页

## ❓ 常见问题

### Q: 访问 `/admin/blog` 显示404？
A: 这是正常现象，表示用户未登录，系统会自动重定向到登录页。

### Q: 登录后还是无法访问？
A: 检查用户角色是否为 admin 或 editor。

### Q: 验证码错误？
A: 刷新页面重新获取验证码，确保在有效期内输入。
